Adds a rich text (WordPress Editor / TinyMCE) widget to your WordPress web site.

== Description ==

Rich Text Sidebar Widget adds a rich text WordPress Editor widget to your WordPress installation, 
allowing you to create one or more sidebar widgets, each with a rich text / visual editor.

The editor works in the same way as the WordPress Posts or Pages Editor, allowing you to:

* Define formatting and font weight,
* Created bulleted and numbered lists,
* Alignment of text,
* Create links
* â€¦ and all of the standard WordPress Editor functions.

== Installation ==
1. Download the Plugin.
2. In your WordPress Administration, go to Plugins > Add New > Upload, and select the plugin ZIP file.
3. Activate the Plugin
